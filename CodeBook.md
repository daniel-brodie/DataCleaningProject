

#Overview
This data file (analysis.data.csv) was produced using raw data described in the included file (Readme-OriginalData).

The data file consists of the following comma delimited fields:
*Subject - numeric - the number that identifies the subject
*Activity - numeric - the number that represents the activity performed by the subject
*AD - characters - the description of the activity - see below for details of these activities
*variable - the variable summarized for each user and activity  See below for details
*VarAverage - the average (mean) of the variable for each user and activity

##Generation
See included README file for details on how the data was produced.

## Activities
The activities are one of the following:  

* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING
